# Navish Bahl — TEM Application

A deck-style web app making the case for the **Technical Enablement Manager** role at Glean.

Built with Next.js 14, Tailwind CSS, and Lucide React. Deployable on Vercel with zero config.

## Sections

| Tab | What it covers |
|-----|---------------|
| **Why TEM?** | Three initiatives: World-Class Onboarding, Enterprise Brain Demo Platform, Domain Expertise Certifications |
| **Why Now?** | Three reasons: Glean's shipping velocity, strategic customer depth gap, AI industry pace |
| **Why Me?** | Three strengths: field experience, passion for teaching, institutional knowledge |

## Run locally

```bash
npm install
npm run dev
```

## Deploy

Push to GitHub and import at [vercel.com/new](https://vercel.com/new) — Vercel auto-detects Next.js, no configuration needed.
